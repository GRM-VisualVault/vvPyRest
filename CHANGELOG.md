# *vvrest* release changelog

## v1.0.0
- `pypi` packaging
- `PEP8` code refactoring
- `unittest` coverage
- supports `python3`

## v1.0.0b1
- supports `python2`
- not a `pypi` package
- supports the following general endpoints: `emails`, `documents`, `files`, 
`folders`, `forminstance`, `formtemplates`, `groups`, `indexfields`, 
`sites`, and `users`.
